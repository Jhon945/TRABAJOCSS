# 📊 Dashboard Administrativo - Actividad 3 CSS3

## 📌 Descripción del proyecto

Este proyecto consiste en el desarrollo de un **Dashboard Administrativo responsivo**, creado como parte de la Actividad 3 del módulo de CSS3.

El objetivo principal fue aplicar técnicas modernas de maquetación web utilizando **CSS Grid**, **Flexbox** y principios de **diseño responsivo**, simulando un sistema de gestión de inventario.

El dashboard permite visualizar información general mediante tarjetas resumen, tabla de datos y un menú de navegación adaptable a diferentes dispositivos.

---

## 🎯 Resultado de aprendizaje aplicado

Se aplicaron estilos avanzados y técnicas de diseño responsivo utilizando CSS3, logrando:

- Organización del layout con CSS Grid.
- Distribución interna de componentes con Flexbox.
- Adaptación del diseño a escritorio, tablet y móvil.
- Implementación de interactividad visual.
- Aplicación de buenas prácticas de accesibilidad.

---

## 🧰 Tecnologías utilizadas

- HTML5 semántico
- CSS3
- CSS Grid
- Flexbox
- Media Queries
- Variables CSS
- JavaScript básico (interacción del menú)

---

## 🧱 Estructura del Dashboard

El sistema está compuesto por:

- **Barra lateral (Sidebar)**  
  Menú de navegación principal del sistema.

- **Header**  
  Encabezado superior con título y menú hamburguesa para móviles.

- **Área principal**  
  Contiene:
  - Tarjetas resumen
  - Tabla de productos recientes

- **Footer**  
  Información general del proyecto.

---

## 📱 Diseño Responsivo

El dashboard se adapta automáticamente según el tamaño de pantalla:

### 🖥️ Escritorio
- Sidebar visible permanentemente.
- Distribución completa en Grid.

### 📲 Tablet
- Sidebar reducido.
- Ajuste automático del contenido.

### 📱 Móvil
- Sidebar oculto.
- Menú hamburguesa desplegable.
- Cierre automático del menú al seleccionar opción o tocar fuera.

---

## ♿ Accesibilidad implementada

- Uso de etiquetas semánticas HTML5.
- Roles ARIA (`navigation`, `main`, `contentinfo`).
- Navegación clara y organizada.
- Buen contraste de colores.
- Elementos interactivos accesibles.

---

## 🎨 Decisiones de diseño

Se eligió un diseño tipo dashboard moderno inspirado en paneles administrativos actuales:

- Colores sobrios para facilitar la lectura.
- Tarjetas con efecto hover para mejorar la experiencia visual.
- Layout limpio y organizado.
- Interfaz adaptable a dispositivos móviles.

El menú lateral desplegable mejora la experiencia de usuario en pantallas pequeñas.
